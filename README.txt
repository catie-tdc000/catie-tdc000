Project Overview

This portfolio website is designed to highlight my artistic expressions, particularly focusing on themes of resilience and transformation through art. The site is built using HTML and CSS, providing a clean and responsive layout.
Features

Landing Page: A welcoming introduction with a poetic message about the power of art.
Gallery: A section displaying various artworks with images that can be viewed in a lightbox.
Contact Section: Information on how to reach me for inquiries or collaborations.

Usage

Click on the navigation links to jump to different sections of the portfolio.
In the gallery section, click on the images to view them in a larger format using the lightbox feature.
Use the contact section to reach out via email for any inquiries.
